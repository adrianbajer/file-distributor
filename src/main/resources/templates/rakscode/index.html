File Distributor


Application for searching and downloading files. Designed for use in the cartography department in one of the major tourist publishing houses.
It facilitates management of the file flow in publishing process.


How it works?

Files are marked by a code called RAKS code. Given a RAKS code, application diplays a form where user selects her or his name and type of job. Basing on user's input
and data from excel file (for testing purposes data are saved in the application code) application finds corresponding files, copies them to selected directory
or displays their data. Additionally it creates an excel file where necessary data are saved (who and for what purpose has downloaded files).
